Code
SEMESTER
├─ id (primary key, auto)
├─ user (FK to User) ← Who owns
├─ name (CharField max 100)
├─ start_date (DateField)
├─ end_date (DateField)
├─ created_at (DateTimeField auto)

Relationship: One user has many semesters

Unique: User can't have two semesters with same name
